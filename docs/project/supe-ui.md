# supe-ui 组件库

### 简介
集NPM包开发、组件开发、文档编写于一体的个人学习项目，同时收集了后台管理系统开发中常见的业务组件，需要时可拿来即用。

- 常见的手写业务组件，拿来即用
- NPM包开发、发布，Webpack打包JS、Gulp打包CSS
- 组件开发思想及过程
- 使用VuePress编写文档，结合GitHub Pages搭建组件文档
- 熟悉Markdown文档编写
