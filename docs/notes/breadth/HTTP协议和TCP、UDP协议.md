
# HTTP协议和TCP、UDP协议

### 网络协议
- HTTP 是应用层，TCP UDP 是传输层
- TCP 有连接，有断开，稳定传输
- UDP 无连接，无断开，不稳定传输，但效率高

### TCP协议
- 有连接（三次握手）
- 有断开（四次挥手）
- 稳定传输

### UDP协议
- 无连接，无断开
- 不稳定传输，但效率高
- 使用场景：视频会议、语音通话
