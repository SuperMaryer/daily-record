    # 一、标题

    Markdown 标题有两种格式。

    1. 使用 = 和 - 标记一级和二级标题

    示例:

    我是1级标题
=================

我是2级标题
-----------------

    效果:

    # 我是1级标题

    # 我是2级标题

    1. 使用#号标记

    示例:

    # 1级标题
## 2级标题
### 3级标题
#### 4级标题
##### 5级标题
###### 6级标题

    效果:

    # 1级标题

    # 2级标题

    # 3级标题

    # 4级标题

    # 5级标题
    # 6级标题

    # 二、段落

    用空行表示, 也可以在要换行的文字后面使用>=2个空格

    示例:

    空行表示

要换行的文字后面使用>=2个空格

    效果:

    空行表示

    要换行的文字后面使用>=2个空格

    # 三、字体

    可以使用常规、斜体、粗体、粗斜体字体：

    示例:

    正常字体
*斜体字体*
_斜体字体_
**粗体字体**
__粗体字体__
***粗斜体字体***
___粗斜体字体___

    效果:

    斜体字体 斜体字体 粗体字体 粗体字体 粗斜体字体 粗斜体字体

    # 四、分割线

    在一行中用三个以上的星号、减号、底线来建立一个分隔线，行内不能有其他东西。你也可以在星号或是减号中间插入空格。

    示例:

    ***
* * *
*****
- - -
----------

    效果:

    # 五、删除线

    段落上的文字若要添加删除线，在文字的两端加上两个波浪线 ~~ 即可

    示例:

    Hello World & 
~~Hello World~~

    效果:

    Hello World & Hello World

    # 六、下划线

    可以通过 HTML 的<u>标签来实现下划线

    示例:

    不带下划线文本 & 
<u>带下划线文本</u>

    效果:

    不带下划线文本 & 带下划线文本

    # 七、脚注

    文本的补充说明，类似tooltip/title。格式为[^要注明的文本]

    示例:

    要注明的文本：[^脚注名]。

[^脚注名]: 脚注文本的补充说明

    效果:

    要注明的文本：^脚注名。
