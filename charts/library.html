                  # 图表库

                  # Echarts

                  关键词：Canvas、Zrender、开源、纯JS、兼容性好、智能响应式、酷炫动画、移动端、简洁友好、好看、类型不全

                  官网：Echarts (opens new window)
                  Github：Echarts Github (opens new window)
                  社区：gallery (opens new window) makeapie (opens new window) ppchart (opens new window)

                  # Chart.js

                  关键词：基于HTML5、canvas、开源、体积小、兼容性好、智能响应式、酷炫动画、移动端、简洁友好、好看、类型不全

                  官网：chart.js (opens new window)
                  中文文档：chart.js中文 (opens new window)
                  Github：chart.js github (opens new window)

                  # Highcharts

                  关键词：基于HTML5、轻量级、开源、分版本付费、兼容性好、图表界的jQuery、少量3D、移动端

                  官网：Highcharts (opens new window)
                  中文文档：Highcharts中文 (opens new window)
                  Github：Highcharts Github (opens new window)
                  其他：技术社区 (opens new window) 第三方拓展 (opens new window) 付费定制 (opens new window) 在线托管平台 (opens new window)

                  # D3

                  关键词：灵活、SVG绘制、开源、JS、更底层、更自由、运行快、免费、学习门槛较高、开发成本较高

                  官网：D3 (opens new window)
                  中文手册：D3中文手册 (opens new window)
                  Github：D3 Github (opens new window)
                  其他链接：gallery (opens new window) Mike Bostock 的示例 (opens new window) D3图表 (opens new window)
                  基于D3的一些库: Plotly.js (opens new window) taucharts (opens new window)

                  # C3

                  关键词：基于D3、开源、免费、JS、自在、可订制、可控的、更自由、上手较难、可重用

                  官网：C3 (opens new window)
                  中文文档：C3中文文档 (opens new window)
                  Github：C3 Github (opens new window)
                  其他链接：demo (opens new window) Mike Bostock 的示例 (opens new window) D3图表 (opens new window)

                  # NVD3

                  关键词：基于D3、开源、免费、JS、可复用、美观、简洁、更自由、上手较难、可重用

                  官网：NVD3 (opens new window)
                  Github：NVD3 Github (opens new window) 其他链接：demo (opens new window)

                  # G2

                  关键词：基于图形语法、开源、免费、JS、数据驱动图形、双引擎渲染、TS、渐进式使用、D3生态、移动端适配

                  官网：G2 (opens new window)
                  Github：G2 Github (opens new window) 其他链接：gallery (opens new window) 实用小笔记 (opens new window) g2理解 (opens new window)

                  # BizCharts

                  关键词：基于G2、开源、免费、React、更灵活、高自定义、强扩展性、定制

                  官网：BizCharts (opens new window)
                  Github：BizCharts Github (opens new window) 其他链接：v3.5.x gallery (opens new window) V4.0.0 gallery (opens new window)
